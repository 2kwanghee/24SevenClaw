## 목적
로컬 실행에 필요한 API 키 및 환경변수를 입력. 입력값은 `.env` 파일로 ZIP에 포함.

---

## 레이아웃

```
┌────────────────────────────────────────────┐
│ ℹ️  이 값들은 로컬 .env 파일에 저장됩니다.  │
├────────────────────────────────────────────┤
│                                            │
│  KEY                    VALUE              │
│  ┌────────────────┐ ┌──────────────────┐   │
│  │ ANTHROPIC_API… │ │ sk-ant-…         │   │
│  └────────────────┘ └──────────────────┘ ✕│
│  ┌────────────────┐ ┌──────────────────┐   │
│  │ OPENAI_API_KEY │ │ sk-…             │   │
│  └────────────────┘ └──────────────────┘ ✕│
│                                            │
│  [+ 환경변수 추가]                          │
│                                            │
│  ⚠️  API 키를 공유하거나 커밋하지 마세요.  │
│                                            │
└────────────────────────────────────────────┘
```

---

## 기능 요구사항

- [x] 키-값 쌍 입력 (동적 추가/삭제)
- [x] 행 삭제 버튼 (✕)
- [x] 환경변수 추가 버튼
- [x] `canProceed = true` (선택사항)
- [x] 보안 경고 메시지
- [ ] 선택한 에이전트/스킬별 필수 환경변수 자동 제안
- [ ] VALUE 필드 마스킹 (비밀번호 타입 토글)
- [ ] 환경변수 유효성 검사 (KEY 형식: 대문자+언더스코어)

---

## 구현 노트

- 환경변수는 `envVars: Record<string, string>` 형태로 스토어 저장
- ZIP 생성 시 `.env` 파일에 주입
