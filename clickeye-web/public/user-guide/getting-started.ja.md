# はじめに

ClickEye へようこそ。

## 概要

ClickEye は、AI エージェントによる開発自動化デリバリープラットフォームです。Web デリバリーコンソールでプロジェクトを作成すると、AI Team がタスクを分解し、ドラフト作成・レビューを行って開発を自動化します。

## プラットフォーム構成

- **Web ダッシュボード**: ソリューション設計およびプロジェクト管理
- **エージェントプラットフォーム**: Claude Code、Gemini CLI、Cursor、Codex に対応
- **CLI ツール**: パワーユーザー向けのコマンドラインインターフェース

## クイックスタート

1. ダッシュボードで **新しいプロジェクト** ボタンをクリックします。
2. プロジェクト情報を入力して作成します。
3. プロジェクトの **AI Team** ページで最初のタスクをリクエストします。
4. AI が生成したドラフトを確認し、承認します。
