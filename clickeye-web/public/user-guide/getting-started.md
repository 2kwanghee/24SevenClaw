# 시작하기

ClickEye에 오신 것을 환영합니다.

## 개요

ClickEye는 AI 에이전트 기반 개발 자동화 딜리버리 플랫폼입니다. 웹 딜리버리 콘솔에서 프로젝트를 만들고, AI Team이 작업을 분해·초안 작성·리뷰하여 개발을 자동화합니다.

## 플랫폼 구성

- **웹 대시보드**: 솔루션 설계 및 프로젝트 관리
- **에이전트 플랫폼**: Claude Code, Gemini CLI, Cursor, Codex 지원
- **CLI 도구**: 파워유저를 위한 커맨드라인 인터페이스

## 빠른 시작

1. 대시보드에서 **새 프로젝트** 버튼을 클릭합니다.
2. 프로젝트 정보를 입력해 생성합니다.
3. 프로젝트의 **AI Team** 페이지에서 첫 작업을 요청합니다.
4. AI가 생성한 초안을 검토하고 승인합니다.
