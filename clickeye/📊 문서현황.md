# 📊 문서 현황 대시보드

> Dataview 플러그인 필요. 모든 수치는 `docs/` 프론트매터(`status`/`category`/`last_updated`)에서 **실시간 자동 집계**된다.

## 🚧 손봐야 할 문서 (needs-revision)
docs-sync 훅이 코드 변경 시 `needs-revision`으로 표시한 문서. `/docs-sync`로 현행화하면 `current`로 복귀.
```dataview
TABLE category, last_updated, related
FROM "docs"
WHERE status = "needs-revision"
SORT last_updated DESC
```

## 📂 카테고리별 분포
```dataview
TABLE length(rows) AS "문서 수"
FROM "docs"
WHERE category
GROUP BY category
SORT length(rows) DESC
```

## ✅ 상태별 집계
```dataview
TABLE length(rows) AS "문서 수"
FROM "docs"
WHERE status
GROUP BY status
```

## 🕒 최근 갱신 Top 15
```dataview
TABLE status, category, last_updated
FROM "docs"
WHERE last_updated
SORT last_updated DESC
LIMIT 15
```

## ⚠️ 프론트매터 누락 문서 (정규화 필요)
```dataview
LIST
FROM "docs"
WHERE !title OR !status OR !category
```
