## 목적
ClickEye 딜리버리 플랫폼을 소개하고 로그인/가입을 유도하는 공개 랜딩 페이지. 병렬 딜리버리 콘솔과 인게이지먼트 기반 워크플로우를 시각적으로 전달.

---

## 레이아웃

```
┌──────────────────────────────────────────────────┐
│ [로고] ClickEye           [로그인] [시작하기]     │

│                                                  │
│   딜리버리 중심 AI 개발 자동화 플랫폼             │

│                                                  │
│    [지금 시작하기]  [데모 보기]                    │
│                                                  │
│  ┌─────────────────────────────────────────┐   │
│  │ 병렬 딜리버리 보드 (콘솔 프리뷰)         │   │
│  │ intake→plan→build→review→merge 스테이지 │   │
│  │ 각 직군별 칩으로 실시간 추적            │   │
│  └─────────────────────────────────────────┘   │
```
