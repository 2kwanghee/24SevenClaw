## 목적
개별 프로젝트의 기본 정보를 보여주고 수정/삭제 가능. 하위 탭으로 대시보드/AI팀/계약/인사이트/설정 이동.

---

## 기능 요구사항

- [x] 프로젝트 기본 정보 표시 (이름, 설명, 상태, 생성일)
- [x] 에이전트/스킬 수 배지
- [x] 플랫폼 표시
- [x] 프로젝트 수정 (이름, 설명)
- [x] 프로젝트 삭제 (확인 다이얼로그)
- [x] 탭 네비게이션 (대시보드/AI팀/계약/인사이트/설정)
- [ ] 프로젝트 아카이브/복원
- [ ] 프로젝트 공유 (팀 멤버)
- [ ] 활동 타임라인 위젯

## API 연동

| 메서드 | 함수 | 트리거 | 설명 |
|--------|------|--------|------|
| `GET` | `apiClient.projects.get` | 진입 | 프로젝트 조회 |
| `PATCH` | `apiClient.projects.update` | 수정 저장 | 프로젝트 업데이트 |
| `DELETE` | `apiClient.projects.delete` | 삭제 확인 | 프로젝트 삭제 |
