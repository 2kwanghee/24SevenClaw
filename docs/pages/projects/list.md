## 목적
사용자의 모든 프로젝트를 탐색하고, 각 프로젝트의 딜리버리 현황과 팀 구성을 한눈에 확인할 수 있는 목록 페이지.

## 기능 요구사항

- [x] 프로젝트 카드 그리드 (sm: 1열 / md: 2열 / lg: 3열)
- [x] 검색 (프로젝트명)
- [x] 상태 필터 (전체 / 활성 / 보관)
- [x] 페이지네이션 (페이지당 10개)
- [x] 스켈레톤 로딩
- [x] 빈 상태 — 필터 결과 없음
- [ ] 빈 상태 — 프로젝트 없음 (첫 방문) 일러스트
- [ ] 정렬 옵션 (최신순 / 이름순 / 에이전트 수)
- [ ] 프로젝트 카드 즐겨찾기 고정
- [ ] 빠른 아카이브 (카드 우클릭 컨텍스트 메뉴)
