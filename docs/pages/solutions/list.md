## 목적
진행 중이거나 완료된 솔루션 세션 목록을 보여주고 새 솔루션 생성을 유도.

---

## 레이아웃

```
┌────────────────────────────────────────────────┐
│ 솔루션                    [+ 새 솔루션]         │
├────────────────────────────────────────────────┤
│                                                │
│  ✨ 새 솔루션으로 AI 자동 설계 시작하기         │
│  [솔루션 설계 시작 →]                          │
│                                                │
├────────────────────────────────────────────────┤
│  회사명 / 솔루션 요청       [completed] 2026-04│
│  회사명 / 솔루션 요청       [generating] ...   │
│  회사명 / 솔루션 요청       [failed] 2026-03   │
│                                                │
│  [페이지네이션]                                │
└────────────────────────────────────────────────┘
```
