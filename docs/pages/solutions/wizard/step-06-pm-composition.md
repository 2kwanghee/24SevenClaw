## 목적
선택한 PM의 구성 요소(AI 에이전트, 스킬, 훅, MCP 서버, 플러그인)를 확인하고 진행 여부 결정.

---

---

## 스토리보드

1. 선택된 PM 프로필 정보 표시 (아바타, 이름, 직책, 도메인)
2. 5개 카테고리별 구성 요소 목록 표시
3. 사용자가 구성 검토 후 "이대로 진행" 클릭 → Step 6 이동

---

## 기능 요구사항

- [x] 선택된 PM 프로필 헤더 (아바타, 이름, 직책, 도메인)
- [x] 5개 섹션: AI 에이전트, 스킬, 훅, MCP 서버, 플러그인
- [x] 각 섹션 아이콘 + 색상 구분
- [x] 빈 섹션 "없음" 표시
- [x] `canProceed = true` (항상 진행 가능)
- [x] 버튼 레이블 "이대로 진행" 오버라이드
- [ ] 구성 요소 클릭 시 상세 설명 tooltip
- [ ] 구성 커스터마이징 (요소 추가/제거)
- [ ] PM 다시 선택하기 링크
