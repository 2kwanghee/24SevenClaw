## 기능 요구사항

- [x] 회사 정보 요약 (회사명, 규모, 업종, 솔루션 요청)
- [x] 선택된 프로토타입 표시
- [x] 선택된 PM (아바타, 이름, 직책, 별점)
- [x] 에이전트/스킬 목록 배지
- [x] 플랫폼 표시
- [x] 환경변수 키 목록 (값 숨김)
- [x] "이대로 진행" 버튼 → `finalize` 호출
- [ ] 각 섹션 "수정하기" 링크 (해당 스텝으로 이동)
- [ ] 생성 완료 후 다운로드 가이드 모달
